Exitar: Game Design Document

by Sujeong Yoon & Regis Bafutwabo

INTRODUCTION

This document describes a game called "Exitar", which is designed to be a simple, 2D side-scroller, build to enjoy a real-life exit man character. The game will use sprite-based animation, collision detection, physics, side scrolling, gravity and tiled backgrounds and 2D game techniques.

TECHNOLOGY

Exitar will be developed for web browsers with javascript enabled using Phaser game engine; which is an HTML5 game framework developed by Photon Storm. Apple's GarageBand is an Apple's application that provide Mac and iOS users with a full-featured recording studio just at the tip of their fingers and it will be used for some of the music and sound effects of the game.For the artwork, we will use Tiled Map Editor for creating and editing the maps of each levels, and Piskel will be used for all the spritesheets and sprites in the game.

BACK STORY

Exitar works for this company in charge of finding a way out of any type of worst case scenario that could happen to any environment or building. Exitar specialization is getting through any type of disaster by finding an exit door in a certain amount of time.

OBJECTIVE

Exitar's objective is to find an exit from a damaged buildings or places full of traps in a certain amount of time.Each level will have a different time based on the difficulty of the level.

If Exitar does not find the exit door, or dies before the time given is over, he will be allowed to try again for 2 more times. However, if he finds exit door within the time given, he will go to the next level. Therefore, it is really important for Exitar to find a way out before the time is over since it is his only way to go to the next level.

GAMEPLAY

This is a side scroller 2D game and Exitar, the main character will be able to jump, crouch, and run left or right onto the surfaces and ledges during the game. He will have to pass through a series of obstacles that will try to prevent him from passing the level.

During the game, Exitar has to collect a total of 5 coins in order to buy himself 5 more seconds of the time given. Also, if he is able to catch any of the tokens, it will give him an extra life in case he uses all of his HP.

CONTROLS

This game will be played using both a keyboard and mouse. Once started, use the following:

  • W - Move Up
  • A - Move Left
  • D - Move Right
  • S - crouching

GRAPHICAL USER INTERFACE

The GUI will be composed of:

  • Splash Screen - The splash screen just simply display the logo of the game on a black background when the game is still loading before showing the Game Menu
  • Game Menu - This is the main menu where the user chooses between the following options:
    • Play Game - If the user clicks on this option, the game will start.
    • Levels - Here the user get to choose which level he would like to play.
    • Help - This will display an info screen with the game information and controls
    • About - If pressed the game will display basic informations about the game authors
  • In-Game Menu - This is the menu that is displayed when the user is already playing the game and it includes:
    • Resume - This option will continue the game when it is in pause state
    • Restart Game - This option will restart the game when clicked
    • Levels - This option will display the levels of the game and let the the user change the levels
    • Controls - This option will display the control keys and their functions.
    • Quit - This option will take the user to the Main Menu of the game
  • Winning and Losing - this pop-up windows will appear when the player pass a level or fail to pass a level. If he manages to pass the level, he will have an option of going to the next level or to restart the level. However, if he does not pass the level, he will only have the option of quiting the game which would take him to the main menu or an option of restarting the level.

ARTWORK

All the artwork in the game will be original and they will be composed of:

  • Exitar - Exitar is a full green character who will have the following animations:
    • Running Left
    • Running Right
    • Jumping Right
    • Jumping Left
    • Jumping Up
    • Dying
  • Obstacles - During the game Exitar will meet some deadly obstacles that are there to prevent him to accomplish his mission.Their animations will be added when time permits.
  • Coins - they will be some goldish coins that will help Exitar buy some time. Their animations include:
    • Coin Circling
    • Floating
  • Tokens - These objects will be in a circular form and will look like the logo.They will be a source of life for Exitar. Their animations are:
    • Logo Circling
    • Floating
  • Exitar's Picture - A small picture of Exitar will be shown on the top-left part of the game's canvas and will contains:
    • A Health Bar - It will decrease based on the amount of time Exitar dies, and will increase as he collects objects.
  • Game Environments - They will be designs with Tiled , and will look different based on the level.
  • Entry door - This is the door where Exitar will be coming from when a level starts.
  • Exit door - This is the door where Exitar will be getting out from when he finishes a level.

SOUND EFFECTS

All the sounds effect of the game will be original:

  • Running
  • Collecting Coins
  • Collecting Tokens
  • Dying
  • Winning
  • Collision with Obstacles
